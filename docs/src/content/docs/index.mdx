## 核心特性

<CardGrid stagger>
	<Card title="自动事务管理" icon="pencil">
		乐观锁机制，无死锁风险。支持 Serializable 和 AllowDirtyWhenAllRead 两种事务级别。
	</Card>
	<Card title="多数据库支持" icon="database">
		MySQL、PostgreSQL、MongoDB、Redis、TiKV、DynamoDB、FoundationDB 等多种后端。
	</Card>
	<Card title="分布式架构" icon="globe">
		Provider-Linkd 架构，全球同服支持，内置 Raft 共识和缓存同步。
	</Card>
	<Card title="多平台客户端" icon="code">
		Java、C#、TypeScript、Lua、Unreal 多平台支持。
	</Card>
	<Card title="游戏开发友好" icon="rocket">
		内置 Online 管理、背包、排行榜、定时器、登录队列等游戏常用模块。
	</Card>
	<Card title="热更新" icon="refresh">
		支持运行时类重载和热更新，无需停服。
	</Card>
</CardGrid>
